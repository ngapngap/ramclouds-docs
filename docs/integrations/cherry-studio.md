# Cherry Studio

## Cấu hình

1. **Settings** → **Providers** → **Add Provider**
2. Chọn **Custom OpenAI Compatible**
3. Nhập:

| Field | Value |
|-------|-------|
| Name | Ramclouds |
| API Base URL | `https://ramclouds.me/v1` |
| API Key | `sk-your-api-key` |

4. Thêm models: `gpt-5`, `claude-opus-4-5`, `gemini-2.5-pro`...
