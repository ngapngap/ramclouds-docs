# FAQ

Frequently asked questions about Ramclouds API.

## General

### What is Ramclouds?

Ramclouds is a unified AI API gateway that provides access to multiple AI models (GPT-5, Claude, Gemini, etc.) through a single, OpenAI-compatible API.

### Is Ramclouds free?

Ramclouds uses pay-as-you-go pricing. You only pay for what you use. Check the [pricing page](https://ramclouds.me) for current rates.

### How do I get an API key?

1. Create an account at [ramclouds.me](https://ramclouds.me)
2. Go to your dashboard
3. Generate a new API key

## Technical

### What's the API base URL?

```
https://ramclouds.me/v1
```

### Is it compatible with OpenAI's API?

Yes, Ramclouds is fully compatible with OpenAI's API format. You can use the official OpenAI SDK by just changing the base URL.

### Which models are available?

Currently 28+ models including:
- GPT-5 series (GPT-5, GPT-5.1, GPT-5.2, Codex variants)
- Claude series (Opus 4.5, Sonnet 4.5, Haiku 4.5)
- Gemini series (2.5 Pro, 2.5 Flash, 3 Preview)
- DeepSeek, Qwen, GLM, and more

Use `GET /v1/models` to see all available models.

### Do you support streaming?

Yes, add `"stream": true` to your request body.

### What are the rate limits?

Rate limits depend on your account tier. Check your dashboard for current limits.

## Billing

### How is usage calculated?

Usage is calculated based on tokens:
- Input tokens: Text you send to the API
- Output tokens: Text generated by the model

## Troubleshooting

### Error: "Invalid API key"

- Check your API key is correct
- Ensure you're using `Bearer` prefix: `Authorization: Bearer sk-xxx`
- Verify your account is active

### Error: "Model not found"

- Check the model ID is exactly correct (case-sensitive)
- Use `/v1/models` to list available models
- Some models may be restricted to certain tiers

### Error: "Insufficient credits"

- Add more credits via [@ramclouds_bot](https://t.me/ramclouds_bot)
- Check your current balance in the dashboard

### Slow response times

- Try a faster model like `gemini-2.5-flash` or `claude-haiku-4-5`
- Enable streaming for perceived faster responses
- Check your network connection
