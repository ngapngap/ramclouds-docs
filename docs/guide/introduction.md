# Introduction

## Ramclouds là gì?

Ramclouds là unified AI gateway - truy cập nhiều AI models (GPT-5, Claude, Gemini...) qua một API key duy nhất, tương thích OpenAI format.

## Tính năng

- **OpenAI Compatible** - Dùng OpenAI SDK, chỉ đổi base URL
- **28+ Models** - GPT-5, Claude, Gemini, DeepSeek, Qwen...
- **Pay-as-you-go** - Trả theo token sử dụng

## Bắt đầu

1. Đăng ký tại [ramclouds.me](https://ramclouds.me)
2. Tạo API key
3. Sử dụng với base URL: `https://ramclouds.me/v1`
