# Contact

## Support Channels

### Telegram Bot

For account management, billing, and general support:

**[@ramclouds_bot](https://t.me/ramclouds_bot)**

The bot can help you:
- Add credits to your account
- Check your balance
- Get API key assistance
- Answer common questions

### Website

Visit [ramclouds.me](https://ramclouds.me) for:
- Account management
- API key generation
- Usage statistics
- Billing information

## Response Times

We aim to respond to all inquiries within 24 hours.

## Before Contacting Support

Please check:
1. [FAQ](faq.md) for common questions
2. [API Documentation](api/overview.md) for technical details
3. Your dashboard for account-specific information

## Reporting Issues

When reporting an issue, please include:
- Your request (without the full API key)
- The error message you received
- The model you were trying to use
- Timestamp of the issue

## Feature Requests

Have a suggestion? We'd love to hear from you! Contact us via Telegram with your ideas.
